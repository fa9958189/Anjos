# Anjos Ambiental

Sistema web de gestão ambiental, operacional, comercial, técnica, jurídica e financeira.

## Estrutura inicial

- frontend: aplicação React + Vite + TypeScript.
- backend: API Node.js + TypeScript.

## Rodar localmente

Em um terminal:

    cd frontend
    npm install
    npm run dev

Em outro terminal:

    cd backend
    npm install
    npm run dev

Frontend: http://localhost:5173
Backend health check: http://localhost:3333/health
